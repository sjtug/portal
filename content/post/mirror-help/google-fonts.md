本镜像提供了 Google Fonts 的反向代理。

**使用方法**

```
<link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">
```

改为

```
<link href="https://google-fonts.mirrors.sjtug.sjtu.edu.cn/css?family=Roboto" rel="stylesheet">
```
